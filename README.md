# 🏦 API Bancária Assíncrona com FastAPI

Este projeto consiste na implementação de uma API RESTful assíncrona para operações bancárias básicas, como depósitos, saques e consulta de extrato, utilizando o framework **FastAPI**.

A aplicação segue boas práticas modernas de desenvolvimento backend, incluindo autenticação com JWT, validação de dados e documentação automática com OpenAPI.

---

## 🚀 Funcionalidades

* Cadastro de contas bancárias
* Registro de transações (depósitos e saques)
* Consulta de extrato por conta
* Validação de saldo para operações de saque
* Autenticação e autorização com JWT
* API assíncrona para melhor desempenho

---

## 🛠️ Tecnologias Utilizadas

* Python
* FastAPI
* SQLAlchemy
* Databases (async)
* Pydantic
* JWT (JSON Web Tokens)
